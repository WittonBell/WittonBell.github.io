C语言随机数

2020年10月21日  |  作者: Witton Bell
 | C/C++
 |  448 字  |  1 分钟
 | 阅读

昨天项目中遇到随机数相关的问题，其实并不是问题，只是没有使用对。项目中使用的自己写的类似C语言的rand()的伪随机函数，但是种子设置有点问题，所以出现了两次战斗有绝大部分相同结果的问题。

C语言的rand()函数在使用前需要使用srand()函数来设置随机种子，这样才不会出现相同的随机序列。rand()函数的取值范围为[0,RAND_MAX]，Windows下的RAND_MAX为32767。实际应用中，我们可能需要生成范围大于RAND_MAX的随机数，直接使用rand()函数就无能为力了。

其实把rand()函数生成的数看做一个比例，然后进行缩放即可生成我们想要的任意范围的数。

rand() * 1.0 / RAND_MAX

就是生成一个[0.0，1.0]的数，这个数就是一个比例，只需要对这个比例缩放到我们想要的范围即可：

 1#include <stdlib.h>
 2
 3// 生成一个[x,y]范围的随机数
 4int Rand(int x, int y)
 5{
 6	int b, d;
 7	if (y > x)
 8	{
 9		b = x;
10		d = y - x + 1;
11	}
12	else
13	{
14		b = y;
15		d = x - y + 1;
16	}
17	return (int)(1.0 * rand() * d / RAND_MAX + b);
18}

使用前记得使用srand()函数设置随机种子。
