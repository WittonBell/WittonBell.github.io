详细安装Ubuntu 21.10

由于公司服务器的硬件配置比较老了，使用的HDD硬盘，集Git，SVN，编译，网盘等等功用于一身，使用体验相当不好，经常卡得受不了。双11前几天就建议老板买一台新电脑作为服务器。电脑很快配置好了，但是商家安装系统搞了一个星期（公司一直都是使用的CentOS系统，大家也相对熟悉CentOS，就希望新服务器也安装CentOS系统。但是商家一直说是安装不了CentOS，包括最新的CentOS 8。既然安装不了CentOS，那安装Ubuntu吧，国外很多都使用Ubuntu系统，我把CentOS与Ubuntu的下载链接发给了商家。但是商家却安装了一个Kylin的桌面系统，说是这个系统可以安装，我问Ubuntu是否可以安装，答可以试一下，就这样，折腾了一个星期才送到。） 1T的SSD硬盘让商家系统盘分配200G的空间，其余的分配给/home，到手却只有200G的可用空间，其余的未分配。由于平常使用中对分区使用得比较少，之前又未使用过Ubuntu，虽然尝试使用Cockpit系统来进行磁盘管理，但是默认情况下不能操作逻辑卷（后面的博文 配置与管理Ubuntu 21.05有介绍如何处理），所以选择了重新安装Ubuntu系统，在安装时重新进行分区调整。本文将详细介绍磁盘分区中的细节。由于与CentOS在很多地方体验都不同，后续博文将介绍Ubuntu下的配置。

一、下载安装包

下载最新的 ubuntu-21.10 Server版本，由于腾讯镜像相对比较快一些，这里选择在腾讯镜像下载，并且将其作为更新源（由于服务器版本是live-server版本，有些功能需要在线下载安装，所以最好选择一个网速比较快的更新源）。

二、安装

在安装时，由于无法选择中文安装语言，所以在安装过程中，需要阅读清楚。

1.是否更新安装器

下图所示可以选择不更新，以加快安装，待安装完成后再更新（那时可以从填写的更新源中更新）。

2. 填写安装（更新）镜像地址

下面的界面则是填写镜像地址了，这里很重要，如果你选择安装的功能很多，很可能需要在线下载安装。

这里填写

1https://mirrors.cloud.tencent.com/ubuntu

3. 磁盘分区

由于是新磁盘，所以选择使用整块磁盘，并且使用LVM组，如果想要在安装完成后可以调整空间，一定要选择使用LVM组，并且选择ext4分区格式。

下图是Ubuntu默认的分配情况

这个界面有三部分信息： 第一部分是FILE SYTEM SUMMARY（文件系统总览），图中挂载了两人个文件系统，/和/boot 第二部分是AVAILABLE DEVICES（可用设备），图中是ubuntu-vg(new)，总大小98.996G，注意下面的free space，有49.5G的空间未分配 第三部分是USED DEVICES（已用设备），图中有两个已用设备ubuntu-vg(new)和/dev/sda。ubuntu-vg(new)总大小98.996G，有49.496G挂载到/，加上前面的free space刚好是整个ubuntu-vg(new)的大小。/dev/sda设备才是真正的整个磁盘，会分三个区，如图所示， 第一个区为Grub使用，占用1M 第二个分区为启动分区，挂载到/boot，占用1G 第三个分区为LVM逻辑卷组，占用剩余的所有空间

如果我们想要调整分配情况，比如/我们调整为30G，其余的都挂载到/home目录。 选择已用设备中挂载到/的行，回车，选择Edit

在弹出的对应框中编辑Size为30G，Save即可。

这个对话框中可以编辑名字、格式以及挂载路径，需要注意的是格式Ubuntu默认为ext4，而CentOS默认是xfs。ext4格式在安装好系统后可以缩小空间，如果磁盘还有空闲，是可以扩容的；而xfs则不能缩小。

/设置为30G后，磁盘剩余空间变成了68.996G了。这些空间都是没有分配的，需要分配它，比如/home目录将是各个用户使用的，我们可以将剩余的空间挂载到/home 如图所示，在ubuntu-vg(new)逻辑卷组上回车，在弹出菜单中选择“Create Logical Volume”创建逻辑卷。

  • Name 名字可修改，也可使用默认
  • Size 如果想把所有空间都挂载到/home则可以不填写Size了，否则填写指定大小
  • Format 选择ext4
  • Mount 挂载点选择/home

然后点击Create创建逻辑卷

完成后，如图所示，整个磁盘都利用起来了。

4.功能选择

在下图中根据要求选择想要安装的功能，可能会从线上下载。

确认后就开始安装系统，在安装系统界面可以选择View full log查看全部日志

从安装日志中可以看到Linux firmware 198M需要从安装（更新）源中下载，如果网络比较慢的话，这步可能要等待很久。

安装后成功登录

然后使用lsblk查看磁盘分区情况，使用df -h查看磁盘使用情况，跟我们之前的预期一样。

Ubuntu的默认locale可能是en_US.UTF-8，没有安装中文包，可以使用下面的命令进行安装

1sudo apt install language-pack-zh-hans
